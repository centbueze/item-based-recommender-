The Netflix Recommendation System is a content-based recommendation engine developed to suggest similar titles based on user input. Designed with a focus on simplicity, performance, and professional presentation, the system utilizes natural language processing techniques to derive semantic similarities between Netflix titles and provide intelligent recommendations in real time.

ðŸ”§ Overview

This application is built using a Python-based backend powered by Flask, and a responsive HTML/CSS/JavaScript frontend. It processes user queries and returns a curated list of Netflix shows or movies that are most similar in context to the given input.

ðŸ’¡ Key Features

  • Content-Based Filtering: Utilizes a TF-IDF vectorizer to transform title and genre metadata into numerical representations, and computes cosine similarity to rank similar titles.
  • Intelligent Matching: The system effectively matches partial and misspelled titles by leveraging text normalization and token-based similarity.
  • Real-Time Recommendations: Asynchronous form submission and dynamic rendering enable smooth, immediate feedback for the user.
  • Responsive UI: The user interface is designed with a clean, Netflix-inspired aesthetic and is fully responsive across devices.
  • Robust Error Handling: Displays clear feedback for errors, empty results, or unrecognized input in a user-friendly manner.

ðŸ›  Technical Stack

  • Backend: Python 3.x, Flask (Micro web framework), Scikit-learn (TF-IDF Vectorizer, Cosine Similarity)
  • Frontend: HTML5, CSS3 (Responsive Design), Vanilla JavaScript (Fetch API for asynchronous communication)
  • Data: Netflix Titles Dataset (Kaggle)

ðŸ§  Methodology

  • Preprocessing: Title, type, and genre data are cleaned and concatenated into a single text corpus.
  • Vectorization: TF-IDF is applied to encode the textual metadata into numerical vectors.
  • Similarity Computation: Cosine similarity is used to determine the distance between the query title and the rest of the dataset.
  • Recommendation Output: Top N results are returned with accompanying metadata including content type, genre, and similarity score.
